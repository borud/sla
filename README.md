# SLA

Extremely trivial SLA calculator.

## Install

    go get github.com/borud/sla/cmd/sla

## Usage

    $ sla 99

      Downtime : 1.000000
      Uptime   : 99.000000

      Daily    : 14 minutes 24 seconds
      Weekly   : 1 hour 40 minutes 48 seconds
      Month    : 7 hours 12 minutes
      Year     : 3 days 15 hours 36 minutes
